# Archive Boundary

Purpose: mark the difference between canonical docs and legacy or working notes.

Last verified: 2026-03-17

This folder is the target location for notes that should no longer be treated as first-stop documentation.

Archive candidates usually have one of these traits:

- date-stamped status that has been superseded
- one-off review or bootstrap prompt
- partial exploratory write-up
- implementation notes that have already been folded into canonical docs

Current rule:

- if a doc is linked from [00-Start-Here.md](/home/rv404/RV404-Lab/PRODUCTIVITY/Obsidian/Test-1a/Apps/tinyrealms/docs/00-Start-Here.md), it is canonical
- if not, it should be treated as working material until explicitly promoted or archived
