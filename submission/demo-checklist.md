- [ ] `npm run demo:doctor` passes or only reports expected non-demo-local differences.

- [ ] Braintrust and model keys are set in the active backend environment.
- [ ] `guide.btc` premium flow captured with txid.
- [ ] `market.btc` premium flow captured with txid.

4. Show the premium response and txid.
5. Trigger `market.btc` paid quote.
